    하이퍼바이저(Hypervisor)란?

    의미

    • 호스트 컴퓨터 1대에서 다수의 운영체제를 동시에 실행할 수 있도록 해주는 가상 플랫폼
    • 가상화머신모니터 또는 가상화머신매니저(Virtual Machine Monitor, Virtual Machine Manager, 줄여서 VMM)라고도 불림
    Type-1 and type-2 hypervisors (출처. Wikipedia)

    분류

    • Type-1 (native, bare-metal)
      • 하이퍼바이저가 하드웨어에서 직접 실행
      • 게스트 운영체제는 하드웨어에서 2번째 수준으로 실행
      • 종류
        • Xen
        • XenServer (Citrix)
        • ESX Server (VMware)
        • L4 microkernel (ex. L4 projects)
        • TRANGO (Wikipedia)
        • POWER 하이퍼바이저 (IBM)
        • 하이퍼-V (Microsoft)
    • Type-2 (hosted)
      • 호스트 운영체제에서 실행
      • VM 내부에서 동작되는 게스트 운영체제는 하드웨어에서 3번째 수준으로 실행
      • 종류
        • VMware Server
        • VMware Workstation
        • VMware Fusion
        • QEMU
        • Virtual PC, Virtual Server (Microsoft)
        • Virtual Box (Sun)
        • Parallels Workstation, Parallels Descktop
